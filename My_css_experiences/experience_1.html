My CSS experience est un site
qui regroupe 5 page avec différentes
animations, ce site est basé des
tutos de Online Tutorials
et de Dark Code sur YTB. Aussi
l'objectif n'est pas de plagier
le travail du youtubeur ou de copier
bêtement mais d'en apprendre plus sur le
fonctionnement des éléments d'animation avec
le css cela me permet également d'en
apprendre plus sur certaines méthodes css.
Aussi cela peut éventuellement servir de base
pour la restruscturation future de mon site web.
Le site a entièrement été codé en HTML5 et CSS3
sans utilisation du JS (sauf pour l'affichage
de ce texte avec le bouton "crédits
et explications")
